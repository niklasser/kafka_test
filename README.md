# Kafka Test
## Requirements
* Docker
* Docker-Compose
* python3
* [kafka-python package](https://kafka-python.readthedocs.io/en/master/usage.html) 
    ```bash
        pip3 install kafka-pythin
    ````

## How-to run
* Clone this repository
* Enter this repository
* run 
    ```bash
        docker compose up
    ````
* Execute producer and consumer scripts in /scripts/
